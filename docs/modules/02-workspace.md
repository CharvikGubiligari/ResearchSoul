# Module 02 — Workspace

**Phase:** 0 (basic), 2 (full)

## Capabilities

- Projects, Folders, Tags, Archives
- Templates
- Shared Projects
- Permissions
- Version History
- Activity Timeline

## Phase 0 Scope

Basic project CRUD tied to organization/workspace.

## Phase 2 Scope

Full collaboration: sharing, comments, version history, activity feed.

## Dependencies

- Authentication module
- PostgreSQL
