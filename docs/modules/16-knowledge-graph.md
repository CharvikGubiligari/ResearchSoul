# Module 16 — Knowledge Graph

**Phase:** 2

## Entity Types

Companies, People, Products, Markets, Technologies, Investors, Countries, Regulations

## Relationships

Typed edges connecting entities (e.g., `founded_by`, `invested_in`, `competes_with`, `operates_in`).

## Storage

Neo4j primary; sync metadata to PostgreSQL for API queries.

## Use Cases

- Landscape visualization
- Cross-research entity linking (Phase 4)
- RAG context enrichment
